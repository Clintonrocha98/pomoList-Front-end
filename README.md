# PomoList - Frontend

O frontend do projeto PomoList foi desenvolvido utilizando tecnologias modernas para acelerar o desenvolvimento.

## Framework e Bibliotecas

- TypeScript
- Next.js
- Tailwind CSS
- Zod
- Axios
- Cookies-Next
- Next Themes
